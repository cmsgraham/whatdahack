About the platform

A professional challenge environment for cybersecurity practitioners

Whatdahack CTF is a recurring security challenge platform designed to develop real-world security skills across the Whatdahack community. Challenges span multiple disciplines — from web security to cloud infrastructure, reverse engineering, and beyond.

Multi-discipline challenges

Curated challenges across web, cloud, forensics, scripting, and more — aligned to real security scenarios.

Live scoreboard & rankings

Track your progress in real time. Compete for rankings and demonstrate measurable skills growth.

Team & individual participation

Participate solo or in teams. Collaborate, learn, and contribute to a stronger security culture.

Regular cadence

New challenges released on a weekly basis, keeping content fresh and skill development continuous.

Challenge categories

Security disciplines covered

Challenges span a broad range of security domains, from hands-on exploitation to theoretical security fundamentals.

Web Security

XSS, SQL injection, authentication bypass, SSRF, and modern OWASP vulnerabilities in realistic web targets.

OWASP Top 10 Resources & training

Cryptography

Break and build ciphers, attack RSA and elliptic curves, and defeat flawed protocol implementations.

RSA · AES · ECC Resources & training

Reverse Engineering

Binary analysis, obfuscation, malware logic, and low-level challenge solving using disassembly and dynamic analysis.

Binary · RE Resources & training

Digital Forensics

Log analysis, memory forensics, network capture investigation, and artifact recovery challenges.

DFIR Resources & training

Binary Exploitation

Buffer overflows, ROP chains, format strings, and heap exploitation to gain control of program execution.

Pwn · Shellcode Resources & training

Miscellaneous

Steganography, encoding puzzles, jail escapes, scripting tasks, and creative challenges that reward lateral thinking.

Steg · Scripting Resources & training

OSINT

Find hidden information using only public data — social media, image metadata, domain records, and satellite imagery.

Recon · Intel Resources & training

Cloud / DevOps

IAM misconfigurations, S3 exposure, container escapes, and cloud-native infrastructure attack vectors.

AWS · Azure · GCP Resources & training

Mobile Security

Android APK reversing, iOS analysis, runtime hooking with Frida, and certificate pinning bypass.

Android · iOS Resources & training

Hardware / IoT

Firmware extraction, UART/JTAG interfaces, embedded device exploitation, and protocol analysis.

Firmware · IoT Resources & training
